16.2  The systems setup interface "doIpTables"

Here is the "netFilter" driver: doIpTables which generates the file parameters of the GNU/Linux firewall. It shares its code with getHostName and can be used as follows:
             getHostName  --netfilter
or :          ln  -s  /usr/local/bin/getHostName  /usr/local/bin/doIpTables  ;  doIpTables

This driver can automatically ("/etc/rc.d/..." or "cron") generate "iptables" chains for the "netfilter" service, from LDAP data (if any), according to the MAC address of the current server (where the program is started):
doIpTables -v -U ldap://ldap...

Example of shell-script (/etc/init.d/) for every GNU/Linux firewall of the intranet:
       ...
       # previous setting...
       ...
       TMP=/tmp/.doIpTables.$$
       2>/dev/null ${BIN}/getHostName --netfilter  >$TMP
       [ $? -eq 0 ] && /bin/sh $TMP
       rm -f  $TMP
       ...
       # next setting...
       ...
