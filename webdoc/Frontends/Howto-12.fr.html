12.2 L'interface de paramétrage système "doRsyncdConf"

Voici l'interface de paramétrage de "rsyncd": doRsyncdConf. Elle partage ses sources avec getHostName. Elle peut donc être utilisée ainsi :
                    getHostName  --rsyncdConf
ou bien :          ln  -s  /usr/local/bin/getHostName  /usr/local/bin/doRsyncdConf  ;  doRsyncdConf

Ce programme permet la génération automatique (déclenchée soit par "cron", soit au niveau des procédures de démarrage du système destinataire) du fichier de configuration de "rsyncd" ("/etc/rsyncd.conf") à partir des informations (si elles existent) contenues dans la base LDAP et concernant l'adresse MAC du serveur courant (où la commande est lancée) :
                         doRsyncdConf -v -U ldap://ldap... >/etc/rsyncd.conf

Exemple de shell-script de démarrage à installer (/etc/init.d/) sur chacun des serveurs de l'intranet à administrer :
       ...
       # previous setting...
       ...
       RSYNCDCONFFILE="/etc/rsyncd.conf"
       TMP=/tmp/doRsyncdConf.$$
       2>/dev/null ${BIN}/getHostName --rsyncdConf  >$TMP
       [ $? -eq 0 ] && cat <$TMP >$RSYNCDCONFFILE
       rm -f  $TMP
       ...
       # next setting...
       ...

Exemple d'utilisation de cette interface sur la "base de tests" :

root@me:~/Cdev#  doRsyncdConf  -v --forceMacAddress  11:22:33:44:55:66
multimedia                                            // Here, it's "stderr"
                                                                   // Now, it's "stdout"...

# @(#)[global]
# @(#)description = Multimedia RSYNC Service
syslog facility = syslog

[music]
comment = Musics
list = TRUE
path = /pub/musics
read only = TRUE

[picture]
comment = Pictures
list = TRUE
path = /pub/pictures
read only = TRUE

comment = Videos
list = TRUE

read only = TRUE

# @(#)LdapEasyInterface - End -

12.3 Les sources C++

La dernière version (20110913) des sources communes à "librsyncconfig.so" et "librsyncshare.so" est là.

   - rsyncConfigFrontend s'obtient par la commande SHELL :
                                                                ln [-s] /usr/local/bin/libsambaconfig.so /usr/local/bin/librsyncconfig.so
                                                                ln [-s] /usr/local/bin/frontend /usr/local/bin/rsyncConfigFrontend
   - rsyncShareFrontend s'obtient par la commande SHELL :
                                                                ln [-s] /usr/local/bin/libsambaconfig.so /usr/local/bin/librsyncshare.so
