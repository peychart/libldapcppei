1. Project presentation:

1.1 What is the software suite "GNULdapEasyAdmin" ?

The objective of this project is to provide the system administrator of an intranet some user-friendly tools to centralize the configuration of its server systems by consolidating their settings in a LDAP (Lightweight Directory Access Protocol) database.

Thus, the software (re)installation of a server will be limited to apply necessary packages to provide its hosting services and its configuration will be automatically insured at system startup, according to the MAC address of its first NIC recorded in the LDAP directory.

Subsequent changes made to parameters in this directory will be automatically updated on each hardware platform involved (eg through the CRON daemon).

Two groups make up this suite of software:
  • graphical interfaces for a user-friendly way to implement or modify the configuration of servers and managed services within the LDAP database,
  • drivers to extract the information in the database directory and convert it to standard configuration files, especially for services whose configuration is not natively interfaced with LDAP.
Note: A profile management allows administrator to delegate some management parameters to different areas of expertise like network administrators, system administrators, responsible for the users hotline, ...
