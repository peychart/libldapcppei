NAME

berValue --- libldapcppei is a C++ library under licence GNU version 3 whose topic is to provide a C++ programming interface for easy and safe use of OPENLDAP databases.  

SYNOPSIS

#include <libldapcppei/ldapBerValue.h>

using namespace ldapcppei;

berValue( void );
berValue( const char *strValue );
berValue( const std::string &stringValue );
berValue( size_t length, const byte*value );
berValue( const BerValue *value );
berValue( const BerValue &value );
berValue( const berValue *value );
berValue( const berValue &value );
berValue( byteArea value );  

CONFIGURATION

The g++ option "-std=c++0x" must be set when using this component: "g++ -std=c++0x ... file ... -lldapcppei -lldap ..."  

DESCRIPTION

berValue() provides LDAP data storage.
It is the basic class of the libldapcppei API, which inherits from the class byteArea.
berValue( void )
Creates an empty berValue.
berValue( const char *strValue )
Creates a berValue of the length of the C string strValue and containing its value.
berValue( const std::string &stringValue )
Creates a berValue and initializes its value with the content of the C++ string stringValue.
berValue( const BerValue *value ) or berValue( const BerValue &value )
Allows to create a berValue with the content of the OPENLDAP structure BerValue.
berValue( const berValue *value ) or berValue( const berValue &value )
Allows to create a berValue with the content of an other berValue.
berValue( byteArea value )
Allows to create a berValue with the content of a byteArea object.
 

PUBLIC METHODS

All public methods of the class "byteArea" are inherited, as well as:
- const char*  toStr( void ) const;
Returns a pointer (const char*) to a C string which contains berValue::fromUtf8().c_str().
 

OPERATORS

All public operators of the class "byteArea" are also inherited.
