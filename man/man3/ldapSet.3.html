NAME

ldapSet --- libldapcppei is a C++ library under licence GNU version 3 whose topic is to provide a C++ programming interface for easy and safe use of OPENLDAP databases.  

SYNOPSIS

#include <libldapcppei/ldapSet.h>

using namespace ldapcppei;

ldapSet( void );
ldapSet( ldapSet const *entryDn );
ldapSet( ldapSet const &entryDn );  

CONFIGURATION

The g++ option "-std=c++0x" must be set when using this component: "g++ -std=c++0x ... file ... -lldapcppei -lldap ..."  

DESCRIPTION

ldapSet() provides a class of objects to be used by the class ldapEasyInterace to implement its input/output buffers. It consists of a std::set of entry objects sorted by dn (using "entry& operator<(entry const */&)").
ldapSet( void )
Creates an empty ldapSet.
ldapSet( ldapSet const *value )
Creates an ldapSet with the contents of *value.
ldapSet( ldapSet const &value )
Creates an ldapSet with the contents of value.
 

PUBLIC METHODS

- ldapSet&             clear( void );
Erases all the contents (entries) of the ldapSet. Returns *this.
- ldapSet&              eraseAll( void );
As clear().
- ldapSet&              erase( void );
Erases the current entry of the ldapSet. Returns *this.
- ldapSet&              erase( std::string const &value );
Search the entry with dn == value and delete it if it exists. Returns *this.
- ldapSet&              erase( ldapSet const */&value );
Erases from the ldapSet all the entries contained in value. Sets the current position to ldapSet::begin() and returns *this.
- ldapSet&              eraseNullEntries( void );
Erases all the null entries of the ldapSet. Returns *this.
- ldapSet&              setNULL( void );
Clears the ldapSet. Returns *this.
- bool                  isNULL( void );
Checks if the ldapSet is empty.
- ldapSet&              ldap_mod_op( int value ); entry& setMod_op( int value==LDAP_MOD_BVALUES );
Sets the ldap_mod_op LDAP_MOD_BVALUES|value for all the attributes of all the entries contained in the ldapSet (current position is not changed). Returns *this.
- ldapSet&              setAddAttributes( void );
Sets the ldap_mod_op to LDAP_MOD_ADD for all the attributes of all the entries contained in the ldapSet. Returns *this.
- ldapSet&              setReplaceAttributes( void );
Sets the ldap_mod_op to LDAP_MOD_REPLACE for all the attributes of all the entries contained in the ldapSet. Returns *this.
- ldapSet&              setDeleteAttributes( void );
Sets the ldap_mod_op to LDAP_MOD_DELETE for all the attributes of all the entries contained in the ldapSet. Returns *this.
- ldapSet&              setSearchAttributes( void );
Sets the ldap_mod_op to LDAP_MOD_BVALUES for all the attributes of all the entries contained in the ldapSet. Returns *this.
- ldapSet&              insert( "entry( value )" );
Insert a entry in the ldapSet. (template on constructors of class "entry"). Current position is set to this new entry. Returns *this.
- ldapSet&              insert( entry *value, bool autoDelete );
Insert the entry *value in the ldapSet. If autodelete is TRUE, the original *value is embedded (fast insert) in the ldapSet object and therfore should not be held by the pointer argument: value. So, be sure to avoid all inappropriate deletion of the original value...
- ldapSet&              insert( entry const *values[] );
Insert the values values as entries in the ldapSet object. Returns *this.
- ldapSet&              insert( entry *values[], bool autoDelete );
Insert the entries *values in the ldapSet. The original *value is embedded in the ldapSet object when autodelete is TRUE (fast insert vs copy). Returns *this.
- ldapSet&              insert( ldapSet const *value );
Insert all the entries of *value in the ldapSet object. Returns *this.
- ldapSet&              insert( ldapSet const &value );
Insert all the entries of value in the ldapSet object. Returns *this.
- bool                  gotoBegin( void ); bool gotoFirst( void );
Sets the current position to the first entry of the ldapSet. Returns begin()...
- bool                  gotoFirst( size_t value=0 );
When value==0, like ldapSet::gotoBegin(). Otherwise, sets the current position to the first entry with a entry::dnSize() == value, ldapSet::end() if a such entry does not exist... Returns TRUE if !ldapSet::end(), FALSE either.
- bool                  begin( void ); bool first( void );
Returns TRUE if the current position is on the first entry of the ldapSet, FALSE otherwise...
- bool                  gotoPrevious( void );
Sets the current position to the previous entry of the ldapSet. Returns !ldapSet::end()...
- bool                  gotoNext( void );
Sets the current position to the next entry of the ldapSet. Returns !ldapSet::end()...
- bool                  gotoEnd( void );
Sets the current position on the last entry of the ldapSet. Returns ldapSet::end()...
- bool                  gotoLast( size_t value=0 );
Sets the current position on the last entry of the ldapSet where entry::dnSize() == value. Returns FALSE if such an entry does not exist...
- bool                  end( void );last( void )
Returns TRUE if the current position is on the last entry of the ldapSet, FALSE otherwise...
- ldapSet&              find( std::string const &value );
Search for the entry where dn is value. Current position of the ldapSet will be ldapSet::end() when value is not found...
- ldapSet&              find( entry const */&value );
Search for the entry with the same dn as value. Current position of the ldapSet will be ldapSet::end() if not found...
- ldapSet&              findStartingWith( std::string const &value );
Search for the first entry where dn starts with value.
- ldapSet&              findNextStartingWith( std::string const &value );
Search for the next entry where dn starts with value.
- ldapSet&              findContaining( std::string const &value );
Search for the first entry where dn contains value.
- ldapSet&              findNextContaining( std::string const &value );
Search for the next entry where dn contains value.
- ldapSet&              findEndingWith( std::string const &value );
Search for the first entry where dn ends with value.
- ldapSet&              findNextEndingWith( std::string const &value );
Search for the next entry where dn ends with value.
- entry*                        getEntry( void );
Returns a pointer to the current entry of the ldapSet or to a nullEntry on ldapSet::end().
- std::vector<entry> getEntriesVector( void );
Returns a std::vector with a copy of every entries of the ldapSet.
- size_t                        size( void );
Returns the number of entries contained by the ldapSet.
- bool                  empty( void );
Returns TRUE if the ldapSet is empty, FALSE otherwise.
 

OPERATORS

- ldapSet&             operator=( entry const */&value );
Assigns (*)value (and its current position) to the ldapSet (which is simply cleared if value==nullptr).
- ldapSet&              operator+=( entry const */&value );
Appends (replaces existing) all entries from *value to the ldapSet and returns *this. Current position is begin().
- ldapSet                       operator+( entry const */&value );
Returns a new ldapSet value with all the entries of the two operands.
- ldapSet&              operator-=( entry const */&value );
All common berValues for attributes of the same entries are deleted and all the empty entries are then erased from *this.*this is returned with "begin()" as current position.
- ldapSet                       operator-( entry const */&value );
Returns a new ldapSet where value is (*this -= value).
- ldapSet&              operator-=( std::string const &value );
If an entry with the dn value is found, it is deleted from this entry.
- ldapSet                       operator-( std::string const &value );
Returns a new ldapSet where the entry of dn value is deleted (*this -= value).
- ldapSet&              operator&=( entry const */&value );
All the erValues, attributes and entries not present in (*)value are erased from the ldapSet and *this is returned. Current position is set to begin().
- ldapSet                       operator&( entry const */&value );
Returns a new ldapSet value of the first operand where the berValues, attributes and entries not present in (*)value are removed.
- ldapSet&              operator|=( entry const */&value );
Appends all the entries from *value to the ldapSet (completing existing values) and returns *this. Current position is begin().
- ldapSet                       operator|( entry const */&value );
Returns a new ldapSet value of the first operand where entries not present in (*)value are added.
- bool                  operator==( ldapSet const */&value ) const;
Returns TRUE if ldapSet is the same than the ldapSet (*)value, FALSE otherwise.
- bool                  operator!=( ldapSet const */&value ) const;
Returns !operator( entry const */& )...
- bool                  operator++( void ) const; bool operator++( int ) const;
Increments the current position (entry pointed by ldapSet::getEntry()) of the ldapSet. Returns !ldapSet::end()...
- bool                  operator--( void ) const; bool operator--( int ) const;
Decrements the current position (entry pointed by ldapSet::getEntry()) of the ldapSet. Returns !ldapSet::end()...
- entry*                        operator[]( size_t i );
Set the current position at the entry number i (sorted by "entry& operator<(entry const */&)") and returns ldapSet::getEntry(). The current position is unchanged if value.size()==0.
- entry*                        operator[]( std::string const &value );
Set the current position at the entry of dn value and returns ldapSet::getEntry().
